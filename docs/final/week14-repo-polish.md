## Clarified Material
Updated comment at the top of FlightResultsPage.jsx file in frontend/src/pages to clarify its deprecrated state. Some of the materials in this file were merged into the destination suggestions page to create a pop-up of the flight results page instead of a brand new page.

## Removed Stale Material
The project originally referenced the Amadeus Self-Service API for flight data. This API is deprecated and is no longer accessible for development use. The integration was removed and replaced with a supported API 
(RapidAPI Flights Scraper Sky) to ensure continued access to real-time flight data. This update prevents confusion for developers and aligns the codebase with currently supported services.
